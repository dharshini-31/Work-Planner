# Project Management Pro

A premium, dual-portal project management application built with Flutter and Firebase.  
Designed with a modern UI and dynamic theme system to differentiate Administrators and Users.

---

## Key Features

### Secure Authentication & Onboarding
- Email verification system for secure access
- Dynamic portal selection with real-time theme switching  
  - Admin Mode: Crimson Red and Black UI  
  - User Mode: Sky Blue and Slate UI  
- Glassmorphic landing page

---

### Admin Portal
- Full Task Management (CRUD)
- Assign tasks with deadlines and priorities
- User directory with card-based UI
- Bottleneck and issue management system
- Dashboard analytics for project insights

---

### User Portal
- View assigned tasks (To-Do, In-Progress, Completed)
- Task cards include admin contact information
- Report project bottlenecks
- View issue history and admin resolutions

---

### Profile and Settings
- Editable profile screen
- International phone input with country flags
- Real-time notifications and badges

---

## Tech Stack

- Frontend: Flutter (Dart)
- Backend: Firebase Firestore
- Authentication: Firebase Auth
- UI Design: Material 3 and Glassmorphism
- State Management: StreamBuilder and Stateful Widgets
